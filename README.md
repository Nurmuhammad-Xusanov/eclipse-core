# eclipse-core (nismoloader_v3)

Telegram bot for downloading Instagram media usiong Instaloader

## Requirements
- Python 3.13
- pip
- Linux/macOS/Windows

## Setup

```bash
https://github.com/Nurmuhammad-Xusanov/eclipse-core.git
cd eclipse-core
chmod +x setup.sh
bash setup.sh
```

## Running

After setting up, edit .env and run these commands: 

```bash
source venv/bin/activate
python bot.py
```

Now, you can enjoy from bot without ads for free!
